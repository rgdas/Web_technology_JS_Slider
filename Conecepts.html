About This Slider

This image slider features smooth fade transitions, autoplay (pauses on hover), and responsive design. Built with vanilla JavaScript for controls and CSS for styling and animations.

Key features:

  • Navigation arrows for manual control
  • Dot indicators to jump to specific slides
  • Autoplay every 3 seconds
  • Responsive layout for mobile and desktop
